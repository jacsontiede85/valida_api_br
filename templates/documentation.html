info

Descrição do Endpoint

Este endpoint realiza consultas completas de CNPJ, combinando dados de protestos do CenProt Nacional com informações da Receita Federal. Suporta múltiplos parâmetros para personalizar a consulta conforme suas necessidades específicas.

send Exemplo de Requisição Completa
Headers da Requisição HTTP Headers 
Content-Type: application/json
Authorization: Bearer rcp_sua_api_key_aqui
Body da Requisição (JSON) POST Data 
{
  "cnpj": "62652677000161",
  "protestos": true,
  "receita_federal": true,
  "simples": true,
  "registrations": true,
  "geocoding": true,
  "suframa": false,
  "strategy": "CACHE_IF_FRESH",
  "extract_basic": true,
  "extract_address": true,
  "extract_contact": true,
  "extract_activities": true,
  "extract_partners": true
}
lightbulb

Dica de Uso

Headers: O Content-Type é obrigatório para JSON. O Authorization deve conter sua API Key.
Body: Este exemplo mostra todos os parâmetros disponíveis. Você pode omitir os parâmetros opcionais ou definir apenas os que precisa. O parâmetro cnpj é obrigatório.
